# 🐨 FoodKuala – Online Food Ordering App

## 📱 Overview
FoodKuala is a mobile application that enables users to discover restaurants, browse menus, place food orders, and track deliveries in real-time. The app focuses on speed, convenience, and a smooth user experience.

---

## ✨ Features

### 👤 User Features
- User registration & login (Email, Google, etc.)
- Browse orders by category, rating, or location
- View detailed menus with images and prices
- Add items to cart
- Secure checkout & multiple payment methods
- Real-time order tracking
- Order history & reordering
- Ratings & reviews

### 🚚 Delivery Features (still under construction 🚧)
- Delivery partner dashboard
- Accept/reject orders
- Real-time navigation
- Update delivery status

---

## 🛠️ Tech Stack

### Frontend (Mobile)
- Flutter
- State Management (Bloc)

### Backend
- Firebase

### Database
- Firebase Firestore

### Other Services
- Firebase Cloud Messaging (push notifications)
